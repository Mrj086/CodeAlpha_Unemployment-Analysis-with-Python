### 📊 Exploring Unemployment Trends and the Impact of COVID-19


------------------------------------------------------------------------

## ✨ Project Overview

This project was built as Task 2 of a data science internship series. This project performs **data cleaning, exploration, visualization, and
trend analysis** on unemployment data in India.

### 🎯 Objectives

-   ✅ Analyze unemployment rates.
-   ✅ Explore trends and patterns.
-   ✅ Investigate the impact of Covid-19.
-   ✅ Identify seasonal variations.
-   ✅ Generate policy recommendations.

------------------------------------------------------------------------

## 🛠 Technologies Used

-   Python
-   Pandas
-   NumPy
-   Matplotlib
-   Seaborn

------------------------------------------------------------------------

## 📈 Analysis Performed

### 🔹 Data Cleaning

-   Missing value inspection
-   Duplicate removal
-   Date conversion

### 🔹 Exploratory Data Analysis

-   Statistical summary
-   Average unemployment rate
-   Region-wise analysis

### 🔹 Visualizations

-   Overall unemployment trend
-   Top regions with highest unemployment
-   Rural vs Urban comparison
-   Correlation heatmap
-   Covid-19 impact analysis
-   Seasonal trend analysis
-   Distribution plots

### 🔹 Policy Insights

-   Employment support programs
-   Skill development initiatives
-   Regional economic policies
-   Social protection measures
